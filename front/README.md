# 前段

## 执行
1. 安装npm  
    可在命令行中使用npm命令
2. 打开前端  
    在front文件夹中打开命令行  
    * 编译命令，文件变更后执行
        ```
        npm install
        ```
    * 运行命令，编译后可仅执行此命令
        ```
        npm run serve
        ```
    cmd中会生成一个网站，打开即为前端
3. 打开前端服务端  
    打开前端网站后会报错，需要额外打开前端的服务端，在front/server文件夹中打开cmd
    ```
    node.exe .\MyServer.js
    ```
